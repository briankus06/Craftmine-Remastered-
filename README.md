## About the Server

Craftmine Remastered is a modded Minecraft Roleplay Economy based server that a group of friends started in March of 2026. We have created a custom modpack that makes a realistic, yet fun, version of Minecraft that can create a working economy with a simulated government that can create laws to help (or hinder) the style of how the server can be played; the power is vested in the players to determine how everything is played. Additionally, there are skill trees that can be altered by each individual player to help make their specific play-style more easier to improve.

## Purpose of the GitHub

This repository contains the .zip file that you can download and move into your Mods folder on Minecraft. Since Discord has a 10 MB file size limit for non-nitro users, I have opted to use GitHub to store the file, since it is something I am familiar with and is relatively easy for new users to navigate.
